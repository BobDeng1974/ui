# Mainflux IoT Admin UI based on Angular 8+ and <a href="https://github.com/akveo/nebular">Nebular</a>

![dashboard][dashboard]

![things][things]

![details][details]

## Install

```
cd ui
npm install
npm start
```

Make UI Docker:
```
make docker_ui
```
